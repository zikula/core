Zikula UI Features

You can use Zikula.UI with: 
{pageaddvar name='javascript' value='zikula.ui'}
or enabling the ui parameter to the ajaxheader plugin call: 
{ajaxheader ... ui=true}
  • Tooltips
  • Window - simple dragable window with button to close
  • Dialogs
    • Default dialog with OK and Cancel buttons
    • Alert dialog
    • Confim dialog
    • Form dialog
  • Multiple select
  • Tabs
  • Accordion
  • Panels
  • ImageViewer
